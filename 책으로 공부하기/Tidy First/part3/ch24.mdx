# 경제 이론: 시간 가치와 선택 가능성

#### 돈의 본성은 2가지 놀라운 속성을 지님.

1. 오늘의 1달러가 내일의 1달러보다 더 가치 있음. ➡️ 버는 것은 빨리하고, 쓰는 것은 늦게 하는 것이 좋음.
2. 혼란스러운 상황에서는 어떤 물건에 대한 옵션이 물건 자체보다 낫기 때문에 불확실성에 맞서는 옵션을 만듦.

소프트웨어 설계는 먼저 벌고 나중에 써야한다와 물건이 아닌 옵션을 만들어야 한다는 2가지 속성을 조화해야 함.

돈의 '시간에 따른 가치'와 '선택 가능성'이라는 2가지 효과를 살펴본 후, SW 설계가 돈과 어떻게 교류하는지 살펴볼 예정
