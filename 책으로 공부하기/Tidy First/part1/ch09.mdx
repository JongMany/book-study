# 설명하는 상수

#### 👍 상징적인 상수로 만들기. 리터럴 상수로 사용된 곳은 상징적인 상수로 변경하기

AS-IS

```
if response.code = 404
  ... 코드
```

TO-BE

```
PAGE_NOT_FOUND := 404
if response.code = PAGE_NOT_FOUND
  ... 코드
```

이 작업을 할 때는 응집도와 결합도를 나중에 신경쓰고, 일단 상수를 만들어서 코드를 이해하기 쉽게 만들자.

- 응집도와 결합도를 신경쓰게 되는 이유: 한 번에 바뀌어야 하거나 함께 이해해야 하는 상수와 그렇지 않는 상수를 분리하는 작업이 필요하기 때문
