# 선언과 초기화를 함께 옮기기

변수 선언과 초기화 위치가 종종 떨어져 있다.

변수 이름을 통해 변수의 역할을 알게되지만, 변수 초기화가 힌트를 극대화해줌

AS-IS

```
fn()
  int a
  ... a를 사용하지 않는 코드
  a = ...
  int b
  ... 변수 a를 사용할 수 있으나 변수 b를 사용하지 않는 코드
  b = ...
  ... 변수 b를 사용하는 코드
```

TO-BE

```
fn()
  int a = ...
  ... 변수 a를 사용하지 않는 코드
  ... 변수 a를 사용하는 코드, 변수 b를 사용하지 않는 코드
  int b = ...
  ... 변수 b를 사용하는 코드
```

초기화를 변수 선언 근처로 이동하여, 문제를 해결함

변수를 사용하기 직전에 선언하고 초기화하는 경우 vs 함수 맨 위에 모든 변수를 함께 선언하고 이후에 초기화하는 경우
➡️ 보통은 변수 사용 직전에 선언하는 것이 더 좋음

이렇게 순서를 바꾸다가 실수로 데이터 종속을 놓칠 수 있음. 이 때, 잘 작동했던 버전으로 돌리면 됨.  
작은 단계로 작업하기! (이것이 올바른 코드 정리임)
