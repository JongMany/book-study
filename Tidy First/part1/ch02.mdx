# 안 쓰는 코드

안쓰는 코드는 지워버리자! 실행되지 않는 코드라면 그냥 지우자

안쓰는 코드를 찾는게 쉬운 경우도 있지만, 쉽지 않을 때도 있음.  
리플렉션을 여러번 사용한 코드가 있을 때, 이 코드가 실행되는지 안되는지 확인하기 어려울 수 있음. ➡️ 로그를 활용해 코드를 지우고 나서 실행해서 확인하기

- 리플렉션: 실행 중인 프로그램인 프로세스가 스스로 구조와 행동을 조사하여 구동 중에 이를 변경할 수 있는 능력

지웠다가 나중에 필요한 경우가 있지 않을까? ➡️ 형상 관리 도구를 쓰면 이런 문제를 해결해줌

단, 각 정리 과정에서는 코드를 '조금만' 삭제해야 함. 그래야 쉽게 복구가 가능함
